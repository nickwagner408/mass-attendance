Setup

Google Sheets Connection

Paste your Google Apps Script Web App URL below. Every volunteer's phone needs this URL entered once.

How to get the URL

  1. Open your Google Sheet
  2. Click Extensions → Apps Script
  3. Delete any existing code and paste the script below
  4. Click Save (floppy disk icon)
  5. Click Deploy → New deployment
  6. Click the gear ⚙ icon and choose Web app
  7. Set Execute as: Me and Who has access: Anyone
  8. Click Deploy and authorize when prompted
  9. Copy the Web App URL and paste it above
